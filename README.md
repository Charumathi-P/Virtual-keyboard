# Virtual Keyboard
A responsive Virtual Keyboard built using HTML, CSS, and JavaScript.

## Features
- QWERTY Layout
- Caps Lock
- Shift
- Backspace
- Enter
- Spacebar
- Dark Mode
- Sound Effects
- Responsive Design
- Physical Keyboard Support

## Technologies
- HTML
- CSS
- JavaScript

## Future Enhancements
- Emoji Keyboard
- Voice Typing
- Multiple Language Support
- Themes
